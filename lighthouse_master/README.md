### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/YOUR_USERNAME/lighthouse_master.git
    cd lighthouse_master
    ```

2.  **Install Dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the App**
    *   **Web**:
        ```bash
        flutter run -d chrome
        ```
    *   **Linux**:
        ```bash
        flutter run -d linux
        ```
    *   **Android/iOS**:
        ```bash
        flutter run
        ```
